# Calculadoras Industriais 🏭

Ferramentas desenvolvidas em Python para auxiliar no controle 
de produção industrial.

## Programas

### 📊 Calculadora de Eficiência (`calculadora.py`)
Calcula o percentual de eficiência e a diferença para bater 
a meta por turno.

**Como usar:**
- Execute o arquivo no terminal: `python calculadora.py`
- Informe o turno, a produção real e a meta
- O programa calcula e exibe a eficiência e as unidades faltantes
- Permite calcular múltiplos turnos em sequência

### 📏 Calculadora de Faixas (`faixas.py`)
Calcula o perímetro de faixas para corte e a metragem total acumulada.

**Como usar:**
- Execute o arquivo no terminal: `python faixas.py`
- Informe a largura e o comprimento de cada faixa
- O programa valida os dados e acumula a metragem total
